##  Project Overview

The **Parking Management System** helps administrators efficiently manage parking spaces by providing:
- Real-time parking status
- Vehicle entry and exit records
- Slot allocation overview
- A centralized dashboard for monitoring operations

This repository contains the **frontend (Angular) implementation** of the system.

- **Language:** TypeScript
